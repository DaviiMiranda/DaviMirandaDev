Projetos

Projeto APP estagiários

Desenvolvi o painel administrativo de um aplicativo mobile voltado para controle de ponto de estagiários. A solução foi construída com React Native no frontend, Node.js no backend e Firebase como banco de dados. O sistema permite o registro de horários, visualização de dados e gerenciamento de solicitações de alteração de ponto, promovendo uma gestão eficiente e transparente para administradores.

Projeto Hotelaria

Criei um sistema completo de gestão hoteleira com funcionalidades como cadastro de hóspedes, controle de reservas, check-in/check-out e atualização automática do status dos quartos. O sistema foi desenvolvido com foco na organização e na eficiência da rotina hoteleira, integrando banco de dados mySQL via API organizada com suas devidas rotas e manipulação SQL .

Projeto Loja Virtual

O projeto reforça minha experiência com consumo de dados em tempo real, organização de front-end escalável, manipulação de dados via Firebase e boas práticas de estrutura semântica em HTML e CSS. Com foco maior no back-end organizei no banco os produtos, e com cadastro de clientes via JWT.
